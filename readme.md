# Untitled Light Desktop Environment (MiniWM)

A simple, lightweight X11-based window manager and desktop environment for Linux.

## Features
- Basic window management with title bars and close buttons
- Draggable windows
- Taskbar with auto-resizing buttons
- Start menu for launching applications
- Autostart support (example: `xterm`)

- ## Notes

- Only tested on ubuntu.
- Start menu support will be fixed later.
